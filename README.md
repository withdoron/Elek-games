# Elek's Kart Racers

A kart racing game by Elek — built in Godot 4.

## How to Run

1. Open `kart-racers/project.godot` in Godot 4
2. Press Play (F5)

## Controls

| Action | Keyboard | Gamepad (Xbox) |
|--------|----------|----------------|
| Accelerate | W / Up Arrow | RT (analog) / A button |
| Brake / Reverse | S / Down Arrow | LT (analog) / B button |
| Steer | A/D / Left/Right Arrow | Left Stick |
| Pause / Menu | Escape | — |

## Current State

One kart, one figure-eight dirt road, rolling hills. The entire focus right now is getting the kart **feel** right. Every dynamic is tunable from the in-game Settings menu — no code changes needed.
